# 人力资源人才管理系统

管理候选人从「招聘 → 入职」的全流程，含 **招聘流程管理、入职办理、人才库、数据看板** 四大模块，支持多位员工同时在线操作、共享同一份数据。

零依赖：仅使用 Node.js 内置模块（`node:http` + `node:sqlite`），无需安装任何第三方包。

## 本地运行

```bash
npm start
```

浏览器打开 http://localhost:3000 即可。

- 默认管理员账号：`admin` / `admin123`
- 新员工注册需要「公司密钥」，默认 `hualian2026`
- 新员工注册后会看到同一份候选人数据（多人协作）

## 环境变量

| 变量 | 说明 | 默认值 |
|------|------|--------|
| `PORT` | 服务端口 | `3000` |
| `COMPANY_KEY` | 新员工注册密钥 | `hualian2026` |
| `ADMIN_PASSWORD` | 管理员初始密码 | `admin123` |

部署到云平台时，在平台的环境变量设置里改掉上面的值即可（尤其是公司密钥和管理员密码）。

## 功能一览

- **招聘流程**：新增候选人 → 简历筛选 → 初试 → 复试 → 终面 → Offer → 入职，可前进/回退
- **人才库**：未录用/储备人才沉淀，带标签与备注，可重新激活
- **入职办理**：标记入职后自动生成入职清单，可勾选、指派办理人、标记完成
- **数据看板**：招聘漏斗（累积式）、录用率、来源分布、本月新增等统计
